Meus trabalhos participativos na Escola

O projeto é uma página web criada com HTML e CSS, destinada a exibir meus trabalhos participativos na escola engenheiro Urbano. O objetivo é criar um ambiente virtual onde seja possível compartilhar meus trabalhos fornecendo uma descrição breve de cada um.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e meus projetos dentro da escola em uma seção flexível (projetos). Cada projeto é destacado em seu próprio contêiner , mostrando uma imagem, detalhes do projeto. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

VER CÓDIGO DO PROJETO
O Mercado de Trabalho e como me adaptar

Este projeto traz uma reflexão sobre o mercado de trabalho atual, que está em constante transformação devido à tecnologia, à globalização e às novas formas de contratação.

Através da página web desenvolvida com HTML e CSS, eu compartilho informações sobre as áreas que mais crescem, as principais exigências das empresas e o que significa ser um profissional competitivo hoje.

Também abordo como posso me adaptar a essas mudanças desenvolvendo habilidades importantes como raciocínio lógico, facilidade com números e, principalmente, capacidade de adaptação algo essencial para enfrentar os desafios de um ambiente profissional em evolução constante.

O site apresenta uma estrutura organizada, com seções dedicadas às soft skills (habilidades comportamentais), ao papel da educação contínua e às estratégias para construir um diferencial profissional desde cedo.

Esse projeto me ajudou a visualizar o que preciso aprimorar para conquistar meu espaço no mercado e reforçou meu interesse por áreas ligadas à análise, desempenho e evolução constante.

VER CÓDIGO DO PROJETO
Minha Escolha Profissional: Educação Física

Este projeto apresenta a faculdade que desejo cursar: Educação Física. A página reúne informações sobre o curso, as áreas de atuação, a grade curricular e as razões que me motivaram a seguir essa profissão.

O objetivo do site é mostrar minha conexão com o esporte, minha paixão por saúde e movimento, e como quero contribuir para o bem-estar das pessoas por meio da atividade física.

Utilizei HTML e CSS para estruturar a página com clareza e leveza visual. O layout inclui seções com curiosidades sobre a profissão, possibilidades de carreira (como professor, personal trainer ou preparador físico), e depoimentos de profissionais da área.

Esse trabalho foi muito especial, pois me permitiu refletir sobre meu futuro e expressar de forma criativa o quanto essa escolha tem a ver com meus valores e objetivos de vida.
